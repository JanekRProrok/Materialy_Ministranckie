# Uroczystość Bożego Ciała

## **Uroczystość Najświętszego Ciała i Krwi Chrystusa**

{% hint style="info" %}
Nie ma wspomnienia tajemnicy dnia w Modlitwach Eucharystycznych.
{% endhint %}
